About my Game Dev Experience

I started to get in to game development at the beginning of my sophomore year of high school. I founded the game development club at my high school, and as a result had to teach myself not only how to code and use unity, but also how to create 2D art, 3D art, rig meshes, animate, create sound effects, and write music. I do my best to put all this together to make some cool games and projects. Most recently I participated in the EGaDS fall game jam and in 30 hours created a game which won the award for best overall game! You can check out that game, Illegal Aliens, by clicking the link below.

My Projects

Smooth Transitioning Portals

This is a project I undertook which originally followed a tutorial, in this short demo you can seamlessly walk through portals. Once I finished the tutorial I realized there was a lot that could be imporved upon, so I spent time making the portals even better. This project heavily used quaternion rotations to not only deal with rotating the player, but also placing them in the proper location after walking through the portals. Definitely worth downloading to try out, can be very disorienting walking through the portals. Click the link below to go to the itch.io page to download the project.

Smooth Portals Project

Procedural Terrain Generation

This project generates a random landscape using a perlin noise map. The black/white values of the noise map are translated into height, which is then stored into a 2D array. This 2D array of heights is used to generate a 3D mesh of the terrain. A texture is then applied by selecting colors for certain ranges of heights. The texture is then applied on to the mesh, creating the landscape the player sees. Click the link below to go to the itch.io page to download the project.

Terrain Generation Project
